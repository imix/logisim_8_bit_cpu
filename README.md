# Logisim 8-bit CPU

This is a toy 8-bit CPU written in [logisim evolution](https://github.com/logisim-evolution/logisim-evolution). 

It has a smallish instruction set.

| command | operation | memory representation |
| ------- | --------- | --------------------- |
| NOP	| 	| 0000 |
| LDA	| Addr -> A	| 0001 |
| LDB	| Addr -> B	| 0002 |
| STA	| A -> Addr	| 0003 |
| STB	| B -> Addr	| 0004 |
| ADD	| A + B -> A	| 0005 |
